# 📚 Что находится в этой папке

Здесь хранится моя личная библиотека и заметки к книгам:

- `Books.md` — список книг, которые я читаю или уже прочитал
- `Quotes.md` — цитаты, идеи и заметки из книг
- `README.md` —это описание раздела 

Как пользоваться:
- Чтобы добавить книгу — открой `Books.md` и вставь запись по шаблону
- Чтобы добавить цитату — открой `Quotes.md` и добавь в нужную книгу
